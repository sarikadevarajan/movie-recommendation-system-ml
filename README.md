# 🎬 Movie Recommendation System using Machine Learning

A modern Movie Recommendation System built using **Machine Learning**, **Python**, and **Streamlit**.

This project recommends similar movies instantly using a content-based recommendation engine with a clean and interactive UI.

---

# 🚀 Features

 Content-Based Movie Recommendation  
 Genre-Based Recommendation  
 Interactive Streamlit Web App  
 Movie Ratings Display  
 Fast Recommendation System using Similarity Matrix  
 Modern Responsive UI  
 Custom CSS Styling  

---

# 🧠 Machine Learning Used

This project uses:

- Content-Based Filtering
- Cosine Similarity
- NLP Feature Vectorization
- Recommendation Engine Logic

The system recommends movies based on similarity between movie metadata such as:
- Genres
- Keywords
- Cast
- Crew
- Overview

---

# 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle
- TMDB Dataset

---

# 📂 Dataset

Dataset used:
- TMDB 5000 Movies Dataset
- TMDB 5000 Credits Dataset

---

# 📸 UI Features

The application includes:
- Dark modern UI
- Interactive movie selection
- Styled recommendation cards
- Responsive design
- Smooth user experience

---

# ⚙️ How It Works

1. User selects a movie
2. The system finds similar movies using cosine similarity
3. Recommendations are displayed with ratings
4. Users can also explore recommendations based on genres
